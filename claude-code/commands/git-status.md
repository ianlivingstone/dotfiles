# Git Status Overview

Display a formatted overview of the git repository status.

## Instructions

Simply run: `~/.claude/commands/git-status.sh`

The script will show:
- Current branch and upstream tracking info
- Staged changes (ready to commit)
- Unstaged changes (modified but not staged)
- Untracked files (not tracked by git)
- Helpful suggestions for next steps
